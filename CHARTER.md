# Stilus Charter

Stilus is a native desktop writing editor.

It exists to provide a lightweight, stable, writer-first text editing environment.

Stilus must remain:

- native
- lightweight
- maintainable
- document-centered
- writer-oriented

Stilus must not become:

- an IDE
- a cloud application
- an AI editor
- an office suite
- a plugin platform
- an Electron application
