# Writing Principles

## Text First

Text creation is the primary purpose of Stilus.

## Writing Before Formatting

Text creation has priority over document presentation.

## UTF-8 First

Documents are stored and handled as UTF-8 text.

## Reliable File Handling

Opening and saving documents must be predictable and safe.

## Keyboard Friendly

Common actions should always be accessible from the keyboard.

## Fast Startup

The editor should remain responsive and lightweight.

## Long-form Friendly

The editor should remain comfortable for articles, essays, research and book-length writing.

## Distraction Free

The interface should support concentration and long-form writing.

## Simplicity Before Complexity

New functionality should only be added when it clearly improves the writing experience.
