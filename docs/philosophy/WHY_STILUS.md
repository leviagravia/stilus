# Why Stilus

Stilus exists to provide a reliable writing environment.

The project focuses on text creation, revision and long-form writing.

Stilus prioritizes clarity, stability and concentration over feature accumulation.

Its purpose is to help writers create text with minimal distraction and predictable file handling.

Stilus is not intended to be an office suite, desktop publishing system or integrated development environment.

It is a writing-oriented text editor.
