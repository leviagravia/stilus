# Ted

## Role in Stilus

Ted is relevant because it targets writing and long-form documents more than programming.

It is not a direct base because its document model is word-processor oriented.

## Strengths

- writer-oriented
- document statistics
- bookmarks
- references
- insert-file workflows
- navigation for long documents

## Weaknesses

- RTF/document processor model
- not a plain text editor
- heavier conceptual scope
- page/layout concerns incompatible with Stilus core

## Relevant Models

- statistics
- bookmarks
- references
- insert file
- long-document navigation

## Things Not To Copy Blindly

- RTF engine
- word processor model
- layout/printing architecture
- embedded rich document assumptions

## Stilus Policy

Use Ted as inspiration for the editorial layer, not the core.
