# Leafpad

## Role in Stilus

Leafpad represents the minimal classical GTK text editor philosophy.

It is important as an identity and simplicity reference.

## Strengths

- minimal
- lightweight
- understandable
- classical desktop editor model
- low feature creep

## Weaknesses

- very limited feature set
- too minimal as direct Stilus foundation
- many modern/editorial features absent

## Relevant Models

- simple file lifecycle
- minimal UI
- close/save/discard/cancel logic
- lightweight editor identity

## Things Not To Copy Blindly

- excessive minimalism
- lack of editorial tools
- lack of modern project governance

## Stilus Policy

Use Leafpad as a simplicity benchmark.
