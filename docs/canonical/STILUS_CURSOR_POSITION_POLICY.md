# STILUS CURSOR POSITION POLICY

Status: CANONICAL

## Purpose

Provide consistent cursor position information.

---

## Line Numbering

Line numbers are 1-based.

First line:

Ln 1

---

## Column Numbering

Column numbers are 1-based.

First character position:

Col 1

---

## Status Bar Format

Ln <line> Col <column>

Example:

Ln 42 Col 17

---

## Design Principles

Writer-oriented.

Human-readable.

Consistent with common text editors.
