====================================================

The right-side Research Panel contains four items:

Tags
Source Notes
Scratchpad

====================================================
COMMAND SEMANTICS
====================================================

Add:

Add current selection or user-entered text to the relevant Research Panel item.

Insert:

Insert selected stored item into the document at cursor position.

Delete:

Delete one selected stored item.

Clear:

Clear the content of a singleton container or all scratchpad content.

Manage:

Open a management dialog or panel for viewing, editing, deleting and organizing stored items.

Scratchpad uses Clear because it is a singleton working area.

References, Tags and Source Notes use Delete because they contain multiple items.

====================================================
LOW RISK
====================================================

Add Reference

Reason:

Can initially add current selection to an in-memory list or simple store.

----------------------------------------------------

Insert Reference

Reason:

Inserts stored text into GtkTextBuffer at cursor.

Undo handled by existing insert-text signal.

----------------------------------------------------

Add Tag

Reason:

Stores short textual tag.

No text buffer mutation unless inserted.

----------------------------------------------------

Insert Tag

Reason:

Inserts selected tag text into GtkTextBuffer at cursor.

----------------------------------------------------

Add to Scratchpad

Reason:

Can append selected text or user text to scratchpad storage.

----------------------------------------------------

Insert Scratchpad Text

Reason:

Inserts scratchpad text into GtkTextBuffer at cursor.

----------------------------------------------------

Clear Scratchpad

Reason:

Clears singleton scratchpad content.

No file lifecycle impact.
