STORAGE POLICY

Storage:
existing GKeyFile configuration system

Configuration group:
[favorites]

Maximum favorites:
20

Keys:
file1 ... file20

RULES

Favorites are file bookmarks.

Favorites are not:
- document-position bookmarks
- cursor bookmarks
- outline bookmarks
- session recovery

Duplicates:
forbidden

Adding an existing favorite:
ignored

Removing a favorite:
compacts the list

Clear Favorites:
removes all entries from [favorites]
