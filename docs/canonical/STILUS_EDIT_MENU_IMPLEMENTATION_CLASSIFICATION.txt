====================================================
LOW RISK
====================================================

Go To Line

Reason:

The status bar already tracks line and column using GtkTextIter.
Implementation requires a dialog, get_iter_at_line, cursor placement
and scrolling.

No undo impact.
No file lifecycle impact.
No persistence impact.

----------------------------------------------------

Duplicate Line

Reason:

Line and offset handling already exist.
Undo records inserts automatically through insert-text signal.

----------------------------------------------------

Uppercase

Reason:

Local text transformation.
No persistence.
No file lifecycle.
Undo handled through text buffer mutation.

----------------------------------------------------

Lowercase

Reason:

Local text transformation.
No persistence.
No file lifecycle.
Undo handled through text buffer mutation.

----------------------------------------------------

Remove Trailing Spaces

Reason:

Local text cleanup.
No persistence.
No file lifecycle.
Undo handled through delete/insert signals.
