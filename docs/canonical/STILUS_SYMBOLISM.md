# Stilus Symbolism

## The Name

The name Stilus derives from the ancient writing instrument used to inscribe text.

It represents deliberate writing, revision, and permanence.

## The Symbol

The project symbol consists of:

- a stylus
- three marks

## The Stylus

The stylus represents:

- writing
- authorship
- craftsmanship
- intentional creation

The stylus is a tool.

Stilus is also a tool.

## The Three Marks

The three marks represent the lifecycle of written work.

Primary interpretation:

1. Writing
2. Revision
3. Memory

Writing creates text.

Revision improves text.

Memory preserves text and knowledge.

## Engineering Interpretation

The three marks also represent:

1. Problem
2. Understanding
3. Solution

This reflects the Stilus engineering method.

## Editorial Interpretation

The three marks may also be understood as:

1. Draft
2. Edit
3. Publish

## Project Meaning

The symbol represents a writing tool that leaves a durable mark.

The purpose of Stilus is not merely producing text.

Its purpose is helping writers create, refine, and preserve their work.
