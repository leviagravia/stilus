# STILUS SIMPLEX — Canonical Features

Version: Simplex v1

## Scope

Stilus Simplex raccoglie esclusivamente funzionalità:

- writing-oriented
- academic-writing-oriented
- desktop-native
- a bassa difficoltà tecnica
- ad alto valore editoriale

Le funzionalità devono produrre un beneficio concreto per la scrittura.

## Multi Window

Stilus supporta finestre indipendenti.

Esempio:

Window A:
documento 1

Window B:
documento 2

Window C:
documento 3

Le finestre possono essere affiancate dal window manager.

## Simplex Boundary

Se una feature richiede:

- architettura complessa
- infrastruttura dedicata
- database interno
- motore separato

allora appartiene a:

- Stilus Plus
oppure
- Stilus Ultra

e non a Stilus Simplex.
