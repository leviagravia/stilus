====================================================
FIRST SOURCE GAP CONCLUSION
====================================================

The current source already provides a stable lightweight editor core.

The biggest architectural gap is not file lifecycle or undo/redo.

The biggest gap is menu migration:

1. remove / absorb Search top-level menu
2. remove Options top-level menu
3. move visual options into View
4. populate empty canonical menus incrementally
