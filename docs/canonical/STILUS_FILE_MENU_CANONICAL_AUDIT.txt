CURRENT AMBIGUITY RESOLVED

Existing command:

File -> Close

Existing shortcut:

Ctrl+Q

Canonical decision:

Close Document and Quit Application must both exist.

Ctrl+Q remains assigned to the command it already belongs to.

The other command will receive a new keyboard shortcut later.

BOOKMARKS DECISION

Bookmarks means:

file bookmarks

Not:

document-position bookmarks
internal anchors
cursor bookmarks
outline bookmarks

Difficulty:
