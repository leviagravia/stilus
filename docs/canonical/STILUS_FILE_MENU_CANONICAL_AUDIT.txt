CLOSE / QUIT IMPLEMENTATION FINDING

Status:
CANONICAL

Current implementation:

File -> Close
Ctrl+Q
airpad_file_close()

Actual behavior:
Quit Application

Evidence:
airpad_file_close() prompts for save if needed and then calls g_application_quit().

Canonical decision:
The current File -> Close command must be renamed conceptually to Quit Application.

Close Document must be implemented as a separate command.

Shortcut policy:
Ctrl+Q remains assigned to Quit Application.

Close Document will receive a separate shortcut later.

Implementation status:
Quit Application behavior already exists.
Close Document is not implemented.

Risk:
Quit Application rename / relabel = LOW
Close Document = MEDIUM, because it must clear the current document safely without quitting the application.

BOOKMARKS POLICY REVISION

Status:
CANONICAL REVISION

Previous decision:
Bookmarks = file bookmarks

Revised decision:
Bookmarks = labeled document-position bookmarks

Reason:
File-level bookmarking is already covered by:

Recent Files
Favorites

Therefore Bookmarks must serve a distinct writing-oriented purpose.

Canonical meaning:
Bookmarks mark important positions inside the current document.

Use cases:
- Add citation here
- Verify source
- Rewrite paragraph
- Expand section
- Check argument
- Add footnote
- Review conclusion

Canonical menu structure:

File
└── Bookmarks
    ├── Add Bookmark...
    ├── Next Bookmark
    ├── Previous Bookmark
    ├── Remove Current Bookmark
    ├── Manage Bookmarks...
    └── Clear All Bookmarks

Bookmark data:
- document path
- cursor position or line reference
- label
- timestamp

Difficulty:
MEDIUM

Implementation status:
Not authorized yet.

Policy:
Bookmarks are editorial navigation markers, not file favorites.

BOOKMARK LABEL POLICY

Status:
CANONICAL

Bookmark display format:

Bookmark #N
Line X
Label

Required fields:

Bookmark number
Line reference
Long descriptive label

Label policy:

The label is mandatory.

The label is not limited to 3 or 4 words.

The label must support longer editorial instructions, for example:

- Add citation here from Augustine / Confessions
- Rewrite this paragraph because argument is weak
- Verify source before final revision
- Expand this section with bibliography note
- Check whether this claim needs a footnote

Purpose:

The label tells the writer how to use the bookmark during revision.

Canonical bookmark meaning:

Position
+
Line reference
+
Editorial instruction label

Implementation status:
Not authorized yet.
